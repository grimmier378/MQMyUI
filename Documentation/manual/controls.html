Commands & Controls

A quick reference for the typed commands and the mouse interactions you'll use most.

The /myui command

CommandWhat it does
/myui or /myui helpList the available commands
/myui listList every window and whether it's currently shown
/myui <window>Toggle a window on or off (for example /myui Player)
/myui show <window>Force a window to show
/myui hide <window>Force a window to hide
/myui settingsOpen the Settings window
/myui themeOpen or close the Themes editor
/myui theme listList the available themes
/myui theme set <name>Switch to a theme by name
Window names aren't case-sensitive, so /myui player and /myui Player both work. Use /myui list if you're not sure of a window's name.

Window controls

ActionResult
Drag the title barMove the window
Drag an edge or cornerResize the window
Click the XClose the window
Lock toggle (Settings → Windows, or the bag pop-out lock icon)Stop a window from being moved or resized
Press Esc (in a pop-out bag or container)Close that pop-out

Some windows (such as Inventory and iTrack) can also be opened and closed with a hotkey you set yourself, configured in the Settings window.

Item controls

These work anywhere an item appears — in your bags, in BigBag, or on the paperdoll:

ActionResult
HoverShow the item's tooltip with full details
Left-clickPick the item up (or drop a held item)
Right-clickUse the item
Ctrl + Right-clickOpen the game's built-in item display
Shift + Right-clickOpen MQMyUI's detailed item info window
Right-click a worn slotOpen the equipment swap menu for that slot
Right-click a bagPop that bag open as its own window

Inside the item info window, click a green empty socket (while holding a matching augment) to insert the augment, and click a socketed augment to open its own info. See Inventory & Items.

Spell gem controls

ActionResult
Hover a gemShow what the spell does, with the real numbers
Left-click a gemCast the spell
Right-click a gemMemorize a different spell into that slot

Group and team controls

ActionResult
Click a group/raid memberTarget that member
Right-click a memberOpen the actions menu (come to me, switch to, make leader, remove/block buff)
XP Bars buttons (Min/Less/Mid/More/Max)Set how much experience that character sends to AA
