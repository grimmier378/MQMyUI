Getting Started

This page walks you from loading the interface to having your first windows on screen and arranged the way you like.

Loading the interface

Type /plugin MQMyUI in the game to turn it on. The first time it loads for a character it sets up a fresh, clean layout automatically — you don't need to configure anything to get going.

The Toolbar

When the interface loads, a small Toolbar of buttons appears. This is your home base. On a brand new character, the Toolbar and the floating status HUD are the only things visible — every other window starts hidden so your screen stays clean until you choose what to show.

The Toolbar's buttons open the windows you reach for most:

ButtonOpens
SettingsThe customization window (show/hide windows, styling, themes, copying)
BigBagThe all-in-one item browser
InventoryYour paperdoll and bags
iTrackThe cross-character item tracker
AAThe Alternate Advancement manager
These five windows always start hidden and are meant to be summoned from the Toolbar when you need them. Closing one just tucks it away again — it won't clutter your screen on your next login.

Opening and closing windows

There are three ways to show or hide any window:

  • Toolbar buttons — for the five windows above.
  • The Settings window — the Windows tab has an on/off switch for every window.
  • Typing a command — /myui <window> toggles a window (for example /myui Player). See Commands & Controls for the full list.

To close a window, click the X in its title bar. For most windows this choice is remembered, so the window stays the way you left it next time you log in.

Moving, resizing, and locking

Drag a window's title bar to move it, and drag its edge or corner to resize it. When you have a window where you want it, you can lock it so it can't be moved or resized by accident — look for the lock toggle in the Windows tab of Settings, or the small lock icon on bag pop-outs.

A locked window still works normally — locking only stops you from dragging or resizing it.

  • Want to change colors, sizes, and themes? See Customizing & Themes.
