Core Services

The Core\ directory holds the framework: shared service classes (injected into every module via ModuleContext) plus stateless myui:: free-function helpers. No Core\ file renders a top-level window.

Stateless myui:: helpers

BarEngine (BarEngine.h)

myui::DrawStyledBar(id, pct, BarStyle, cur=-1, max=-1) — ports the Lua bar engine: tween, multicolor gradient fill (4-corner colors + coverage-mask rounding), background, ticks, shimmer, glow, border, centered text. Also exposes DrawStyledBarRect(...) for an explicit screen rect. Consumes a BarStyle. See Bars & Widgets.

Widgets (Widgets.h)

A full imanim-driven, theme-colored widget set (shared verbatim by the sibling MQMy* plugins): DrawToggle, PillTabBar/PillSelectable, BeginAnimatedHeader/BeginAnimatedTreeNode, StyledSliderFloat/Int, StyledCombo/StyledBeginCombo, StyledCheckbox, StyledButton/StyledSmallButton (drop-in for ImGui::Button), OpenInputPopup/InputPopup, StyledEditField, and HelpMarker. Reusable primitives: SoftGlowRoundRect, HoverIntensity, PressScale, DrawHoverShimmer, Soften. SetAnimationsEnabled(bool) is the global motion switch. See Bars & Widgets.

Actions (Actions.h:6)

Deferred game automation, serviced by PulseActions() each pulse: GiveItemTo(spawnId), SwapToWornSlot(...), PickupCoin(...), StartBulkTrade(...), KickFromGroup(spawnId, name), and StartSpellSetLoad(steps)/SpellSetLoadActive(). State machines drive /itemnotify, /nav, native window clicks, and spell-gem loading — so UI code can request an action and let it complete over several pulses. KickFromGroup defers target-setting + /disband until the target matches the intended member. StartSpellSetLoad loads a gem loadout via /memspell per slot with per-slot waits + timeout-skip, clears non-set slots, then stands; consumes a vector of SpellSetStep (:15, {slot, spellId}).

InventoryData (InventoryData.h)

Live inventory reader/renderer. Frame-local readers WornItem/BagItem/BagContents/GetBagContents/GetBank/GetCompatibleItems/CursorItem/GetFreeSlots/FindItemById/FindBagItemByName; predicates IsClicky/IsAugment/ItemFitsSlot; counts CountInventory/CountBank; commands PickupCommand/PickupOrDrop/UseItem; rendering DrawItemIcon/RenderTooltip/RenderCompareTooltip/RenderCompareSideBySide/PopInfoWindow/DrawPoppedInfoWindows/DrawCurrencyRow/DrawCoinQuantityWindow/DrawDropZones; deferred PulseAugInsert (drives augment-slot state machine each pulse). ItemRef (:21) wraps an item pointer + location. The item-info tooltip / popped window, augment slotting, and spell-effect rendering all live here.

AAData (AAData.h:25)

Alternate-advancement reader. Update (rebuild on AA-points change), Rebuild (walk pAltAdvManager->abilities), Available/Spent/Total, CategoryRanks/AllRanks, ByType/Find/All, Description (+ resolved spell effect text), Ready. AAEntry (:6) is one consolidated entry per AA GroupID.

SpellInfo (SpellInfo.h)

myui::SpellEffectText(spellId) returns value-resolved per-slot effect text via MQ's exported ShowSpellSlotInfo (the only source that fills the DB's #1/%z tokens); myui::RenderSpellEffects(spellId) draws a separator + wrapped soft-blue effect block inside any tooltip/window. Shared by item efx, AA descriptions, the spell gems, and the ability picker.

HotkeyCombo (HotkeyCombo.h)

myui::ToggleCombo (key + 3 mods + mouse); ReadCombo(WindowConfig) and CheckToggleCombo (ImGui key + mods, skips while typing) drive per-window show/hide hotkeys.

HPCurrent*100/HPMax : HPCurrent.

PeerData (PeerData.h)

myui::PeerRecord (remote snapshot), myui::ItemCount, GroupRoleBit flags, PeerKey(server, char) key builder. See Types & Constants.

Buff icon renderer (BuffRow.h/.cpp)

myui::DrawBuffIconLineup(const char* idKey, const std::vector<BuffInfo>& buffs, const BuffIconLineupOpts& opts, IconHelper* icons) (BuffRow.cpp:12) draws a line-wrapping row of buff/song icons. Shared by the Buffs window, Songs window, and Player module's target buffs. The caller owns the surrounding table/row/name layout; this renders only the icon lineup at the cursor. Each icon shows an empty-slot placeholder if empty, flash tint on expiry, border color (beneficial/detrimental/self-cast), and on-hover tooltip with caster and duration. Optional contextMenu callback in opts fires on right-click. See BuffIconLineupOpts (BuffRow.h:13) in Types & Constants.
