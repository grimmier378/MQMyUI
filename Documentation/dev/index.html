MQMyUI Developer Documentation

MQMyUI is a native C++ MacroQuest plugin that ports the MyUI Lua UI bundle into a single self-contained DLL. It renders all player / group / raid / inventory / AA UI through ImGui, persists per-character settings in SQLite, syncs live state across boxed characters peer-to-peer with protobuf actors, and is fully customizable (per-window scaling, per-bar styling, global themes).

The architecture is modular: one ModuleBase subclass per window, orchestrated by a ModuleManager, all reading a fixed set of shared services live every frame. There is no build system or test harness here — the plugin is built inside the MacroQuest Visual Studio solution and tested manually in-game.

New to the codebase? Read Architecture first for the big picture, then Module System to learn how to add a window. The other pages are reference.
